# Sistema de Almoxarifado - SENAC Enfermagem

Projeto desenvolvido para a disciplina de ESOFT. O sistema foi criado para ajudar a enfermeira Camila a controlar o estoque de materiais usados nas aulas práticas do curso técnico de Enfermagem.

## Sprint 1

Nessa primeira entrega foi feita a estrutura base da aplicação. O usuário consegue cadastrar materiais novos e visualizar os que já estão no estoque, tudo conectado com a API do MockAPI.

## Tecnologias usadas

- HTML
- CSS
- JavaScript (fetch e async/await)
- MockAPI.io

## Como rodar

Só abrir o arquivo index.html no navegador. Os dados são carregados direto da API.
